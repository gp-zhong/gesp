数组与字符串

数组概念

  • 定义与特性
  • 数组的声明与初始化
  • 数组的操作与遍历

数组是一种用于存储多个相同类型数据的数据结构。在JavaScript中，数组可以保存任意类型的数据。

// 数组声明与初始化
let numbers = [1, 2, 3, 4, 5];
let fruits = ['apple', 'banana', 'orange'];

// 数组访问
console.log(numbers[0]); // 输出: 1
console.log(fruits[1]);  // 输出: banana

// 数组遍历
for(let i = 0; i < numbers.length; i++) {
    console.log(numbers[i]);
}

// 更现代的遍历方法
numbers.forEach(num => console.log(num));

字符串处理

  • 字符串基本操作
  • 常用字符串函数
  • 字符串算法示例

字符串是用于表示文本的数据类型，在JavaScript中是不可变的。

// 字符串创建
let greeting = "Hello, World!";
let name = 'Alice';

// 字符串连接
let message = greeting + " My name is " + name;
let template = `${greeting} My name is ${name}`; // 模板字符串

// 常用字符串方法
console.log(greeting.length);     // 长度
console.log(greeting.toUpperCase()); // 转大写
console.log(greeting.split(', ')); // 分割
console.log(greeting.includes('Hello')); // 包含检查
