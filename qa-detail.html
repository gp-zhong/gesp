问题: 如何理解递归函数?

提问时间: 2023-03-15 14:30
DeepSeek回答:

递归函数是一种调用自身的函数。理解递归需要掌握:

  1. 基本情况(Base case): 不再调用自身，直接返回结果
  2. 递归情况: 将问题分解为更小的子问题

例如计算阶乘:

function factorial(n) {
  if (n <= 1) return 1;  // 基本情况
  return n * factorial(n-1); // 递归情况
}

理解递归的关键是跟踪函数调用栈，理解每一层如何依赖更小问题的解。
