标准答案:

function sum(n) { let total = 0; for(let i = 1; i <= n; i++) { total += i; } return total; }

解析:

这道题考察了循环和累加的基本概念。

  • 我们可以使用for循环从1遍历到n，将每个数字加到total变量中。
  • 也可以直接使用数学公式: n*(n+1)/2 计算。

优化方案:

// 使用数学公式优化 function sum(n) { return n * (n + 1) / 2; }

这种方法时间复杂度为O(1)，比循环的O(n)更高效。
