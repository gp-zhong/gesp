问题1解析

正确答案: D

DeepSeek解析:

选项D是正确的。在JavaScript中，使用let声明的变量具有块级作用域，这意味着它们只在声明它们的代码块内可见。

选项A错误: 全局变量在函数内部可以访问。

选项B错误: 局部变量只在其声明的函数内部可访问。

选项C错误: 在函数内声明的变量默认是局部变量，除非使用var而不使用let/const，并且不是显式声明。

示例代码:

// 块级作用域示例
{
  let blockVar = "我只在块内可见";
  var noBlockVar = "我在块外也可见";
}
console.log(typeof blockVar); // "undefined"
console.log(typeof noBlockVar); // "string"
